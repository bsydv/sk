她意识到，那碗粥用的不仅仅是灵水，就连米还有那个咸菜也都不是凡物！
　　这些东西就和高人一样，表面上看上去是凡人，实则高不可攀。
　　白落霜占了一个大便宜，连忙站起身，惊喜道：“谢，谢谢李公子。”
　　“一碗粥而已，有什么好谢的。”
　　李念凡摇了摇头，这丫头啥都好，就是太容易大惊小怪了，只是一碗粥而已，至于高兴成这样吗。
　　白无尘拿出一个木盒，开口道：“李公子，上次小女从你这里带回去一幅画，我们受益良多，为了表示感谢，这是我们的一点心意，还请千万不要嫌弃。”
　　“你们真是太客气了。”
　　李念凡感到惊喜，来就来了，居然还带了礼物。
　　看来他们是真正的爱画之人。
　　他也没有矫情，打开盒子，却见里面放着一块剑形玉石。
　　这玉石不知道是什么材质，居然如同光源一般，一直在向外散发着白色的光芒，明亮但并不刺目，显得有些神异。
　　林清云用手捂住自己的嘴巴，眼眸难以置信的看着白无尘，内心掀起了惊涛骇浪。
　　剑瑕仙玉，这可是万剑仙宗的镇宗之宝，可以让人领悟剑意成为剑仙的宝贝，白无尘居然就这么拿出来送人了？
　　对于剑修来说，这块玉绝对是无价之宝！
　　白无尘则是面色平静。
　　他也有自己的思量。
　　万剑仙宗本就是李念凡救下的，而且，李念凡随手一幅画的价值可比剑瑕仙玉宝贵多了。
　　剑瑕仙玉对于元婴期的修士来说就已经没有多大的用处了，但李念凡的一幅画，别说元婴期，就算是出窍期、渡劫期乃至仙人都不敢无视。
　　如果说剑瑕仙玉可以让万剑仙宗万年不衰，那李念凡的一幅画就可以让万剑仙宗千万年不衰！
　　我要这剑瑕仙玉还有何用？
　　李念凡从盒子里拿出剑瑕仙玉，放在手里端详。
　　入手冰凉温润，手感上佳。
　　李念凡点了点头，微微一笑道：“这东西不错，刚好可以用来给我晚上照明，当电灯用。”
　　真不愧是高人。
　　剑瑕仙玉居然只是用来照明，不过一想到坠魔剑只是被用来砍柴，他们也就释然了。
